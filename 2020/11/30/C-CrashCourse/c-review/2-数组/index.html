关于static 的思考

对于静态变量的声明应该在static语句中声明

a）一个全局变量被statica修饰，使得这个全局变量只能在本源文件内使用，不能在其他源文件内使用。
extern int a （所要执行的源文件内声明）

static int a （包含a的源文件内声明）

b）一个函数被static修饰，使得这个函数只能在本源文件内使用，不能在其他源文件内使用。
extern int add(int x,int y)
